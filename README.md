# 🎱 FXGL 2D Billiard

Ein klassisches 2D-Billardspiel in Top-Down-Perspektive, entwickelt mit **Java** und der **FXGL Game Engine**.

---

## 🕹️ Gameplay

Ziele mit dem Queue auf die weiße Kugel, reguliere die Stoßstärke und versenke alle Bälle in die Taschen – mit realistischer Physik und Kollisionsberechnung.

---

## ✨ Features

- 🎯 Top-Down-Perspektive auf den Billardtisch  
- ⚙️ Physikbasierte Ballbewegung mit Reibung und Impulsen  
- 💥 Kollisionserkennung zwischen Bällen und Banden  
- 🕹️ Intuitive Steuerung für Richtung und Stoßstärke  

---

## 🛠️ Technologien

| Technologie | Verwendung         |
|-------------|--------------------|
| Java        | Spiellogik         |
| FXGL        | Game Engine & Physik |
